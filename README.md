
# Kitap Listesi Uygulaması

Bu proje, React ve Tailwind CSS kullanılarak geliştirilmiş bir kitap yönetim panelidir.

## 🛠️ Teknik Detaylar
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS (Kurulum hataları nedeniyle CDN üzerinden optimize edildi)
- **Mimari:** Proje; `Components`, `Pages` ve `Interfaces` klasör yapısıyla modüler hale getirilmiştir.

## 📝 Geliştirme Notları
Geliştirme sürecinde terminal üzerinden Git konfigürasyonu yaparken teknik aksaklıklar (git not recognized hatası) yaşadım. Projeyi aksatmamak adına kaynak kodları manuel olarak organize edip GitHub'a aktardım. 

Bu süreçte karşılaştığım hatalar, dosya dizin yapısını ve React'in çalışma mantığını daha iyi anlamamı sağladı.
